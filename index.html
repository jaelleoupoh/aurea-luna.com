Étapes pour publier aujourd'hui
1

Publier le thème Shopistart ⚡ PRIORITÉ ABSOLUE

Shopify Admin → Boutique en ligne → Thèmes → "Shopistart" → cliquer Publier. C'est la seule action bloquante — tout le reste du design est prêt.

admin.shopify.com/store/aurea-luna/themes
2

Configurer la navigation principale

Admin → Navigation → Menu principal. Ajouter : Beauté (collections), K-Beauty, Skincare, Maquillage, Tech & Gadgets, Best Sellers, Nouveautés.

admin.shopify.com/store/aurea-luna/menus
3

Ajouter l'image hero bannière

Admin → Thèmes → Personnaliser → Section Hero → Upload une belle photo beauté/cosmétiques. Sans image, le fond dégradé sombre est déjà en place.

admin.shopify.com/store/aurea-luna/themes → Personnaliser
4

Importer des produits via CJdropshipping & Dsers

Apps déjà installées. Chercher : soins coréens (COSRX, Some By Mi), cosmétiques tendance, gadgets beauté, accessoires tech premium. Important : lors de l'import, ajouter les tags correspondants (k-beauty, skincare, maquillage, etc.) pour que les collections se remplissent automatiquement.

admin.shopify.com/store/aurea-luna/apps
5

Configurer les paiements africains (Wave, Orange Money, MTN)

Installer CinetPay, DPO Group, ou Flutterwave depuis le Shopify App Store. Ces apps activent Wave, Orange Money et MTN MoMo pour la Côte d'Ivoire et l'Afrique.

admin.shopify.com/store/aurea-luna/settings/payments
6

Configurer les marchés internationaux

Admin → Paramètres → Marchés. Activer Côte d'Ivoire, autres pays africains, Europe, International. Configurer le XOF (Franc CFA), EUR, USD pour le multi-devises.

admin.shopify.com/store/aurea-luna/settings/markets
